Comandos para criar a sua Branch, no GitHub

Visualizar o flow das branch: git log --graph
Visualizar as branch criadas: git branch
Criar uma branch: git branch nomeDaBranch
Remover uma branch criada: git branch -d nomeBranch
Selecionar uma branch criada: git checkout nomeDaBranch
